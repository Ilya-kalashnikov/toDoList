# toDoList

Планировщик задач с возможностью добавлять, изменять, удалять задания.

# Установка

1) Клонируем

```
https://github.com/Ilya-kalashnikov/toDoList
```

2) Устанавливаем всё необходимое
```
pip install -r requirements.txt
```

3) Запускаем
```
python manage.py runserver
```
